## Requisito Previos

- Python 3.6 o superior
- Tkinter (Incluido en Python)

1. **Clonación del repositorio**
git clone https://github.com/lmdiaz-dv/supreme-enigma.git
cd tp06-equipo-futbol

2. **Verificación de instalación de Python**
python --version

3. **Tkinter esta incluido en Python versión**
import tkinter

4. **Ejecución de interfaz Tkinter**
python main.py
